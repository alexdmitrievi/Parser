# DNS-записи для привязки домена рег.ру к Vercel

## Корневой домен (yourdomain.ru)

```
Тип:     A
Хост:    @
Значение: 76.76.21.21
TTL:     3600
```

## Поддомен www (www.yourdomain.ru)

```
Тип:     CNAME
Хост:    www
Значение: cname.vercel-dns.com
TTL:     3600
```

## Поддомен (например tenders.yourdomain.ru)

```
Тип:     CNAME
Хост:    tenders
Значение: cname.vercel-dns.com
TTL:     3600
```

## Где добавлять

1. reg.ru → Личный кабинет → Мои домены → [ваш домен]
2. DNS-серверы и управление зоной → Управление зоной DNS
3. Добавить запись → выбрать тип → заполнить поля → Сохранить
